Projetos Programação Tubarões, Raias e Vieiras


Tubarões, Raias e Vieiras



Descrição

O projeto desenvolveu uma modelagem preditiva de populações de Raias, Vieiras e Tubarões e suas relações, utilizando métodos de equações diferenciais em função do tempo no Python.

Competências
  • code Programação
class
Metodologia

O código foi feito em iPython Notebooks e os dados foram coletados de diversas fontes científicas diferentes, principalmente sobre o impacto numérico das relações entre os animais.

build
Técnicas Aplicadas

Foi utilizado um conhecimento sobre equações diferenciais ao longo do tempo, sendo de forma ainda básica, além de conceitos de programação em Python.

grade
Recepção

O projeto mostrou-se sensato e com medidas reais, foi bem avaliado no quesito inovação, já que poucos alunos utilizaram a temperatura do ambiente em sua modelagem.
