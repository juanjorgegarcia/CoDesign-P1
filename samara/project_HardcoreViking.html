Space Track



Descrição

Jogo de exploração espacial para crianças, no qual, o astronauta tinha que recolher informações de cada planeta para depois retornar à Terra. Em cada planeta existia um mini-jogo que apresentava informações sobre ele. .

Competências
  • code Programação
  • create Design
class
Metodologia

O trabalho se dedenvolveu de forma que os professores auxiliares de sua execução faziam checkpoints frequentes semanais, dessa forma era decidido o formato que o programa em Python tomaria.

build
Técnicas Aplicadas

O grupo utilizou de uma biblioteca do Python 3.6 conhecida como Pygame, na qual é possível ter uma base para criação de janelas e objetos em tela.

grade
Recepção

O jogo foi bem recebido pelos professores, porém foi possível observar capacidade de melhoria.
